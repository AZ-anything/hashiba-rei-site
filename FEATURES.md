# ReiVox 機能一覧

## サイト機能（index.html）

### 作品カタログ
- DLsite作品一覧表示（works.jsonから読み込み）
- 発売日の新しい順にソート
- **NEWバッジ**: 発売30日以内の作品に赤パルスアニメーション付きバッジ
- **予告バッジ**: announced/upcoming/preorder作品に表示、発売時期テキスト付き
- **ジャンルタグフィルター**: 上位15ジャンルをタグUIで絞り込み
- **テキスト検索**: タイトル＋ジャンル横断検索
- **はねしば作品識別**: サークル名による自動判定

### YouTube動画セクション
- 動画・歌ってみた・配信・ショートの4タブ切り替え
- タブごとの件数表示
- 初期表示12件 →「もっと見る」ボタンで全件展開
- **メンバー限定バッジ**: 緑色のバッジ（members_only: trueの動画に表示）
- **「メンバーになる」リンク**: YouTubeメンバーシップへの導線
- サムネイル・タイトル・日付表示

### その他
- JSTタイムスタンプ表示
- OGP設定（"Voice Actor"表記）
- レスポンシブデザイン（モバイル対応）
- カスタムドメイン reivox.jp

## 自動化（GitHub Actions: check.yml）

毎日 0:10 JST に実行:

1. **YouTube動画取得** (`fetch_youtube.py`)
   - YouTube Data API v3で最新50件取得
   - タイトルから動画タイプを自動分類
   - ⚠️ **既知の問題**: 50件上限、メンバー限定未対応（後述）

2. **発売チェック** (`check_releases.py`)
   - announced/upcoming/preorder作品の発売状態をDLsiteで確認
   - 発売検知時: status→released、正確な発売日・ジャンル取得
   - 発売時期の変更検知: 「5月」→「5月15日」等の具体化を自動更新
   - 発売時期未設定の作品: DLsite告知ページから自動取得
